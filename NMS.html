Video Game User Interface

My Master's thesis defined a new heuristic framework for video game design and tested it upon an existing video game, No Man's Sky. My new framework took greater account of usability and accessibility concerns during the early phases of prototyping.


No Man's Sky Prototype

UI

For my Master's thesis project, I started by researching UX/UI best practices in the video game industry. What I learned is accessibility and usability are oftentimes forgotten about until a game is nearing release. This led me to define a new heuristic framework for developers to consider before user testing, as well as during the iterative design process as a whole.

Using my heuristic framework, I applied it to No Man's Sky, a space exploration game released in 2016. At its launch, No Man's Sky was criticized heavily for its "unwieldy" and "finicky" controls. My framework was able to identify a number of issues that developers could use to improve the user experience.

I then tackled those issues by redesigning the UI within No Man's Sky. After creating wireframes, I developed an interactive prototype in Figma. I then re-examined my prototype using my heuristic framework to determine if the iterative design process was successful.
